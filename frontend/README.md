# Front-end [React]

### Versions

~~~shell
npm 8.5.5 \
create-react-app 5.0.1 \
node v16.15.0 \
~~~

### Requirements

~~~shell
npm install react-slick
npm install slick-carousel
npm i react-intersection-observer
npm i axios
npm install -s react-toasts
npm install --save react-scroll
~~~

### Start

~~~shell
npm run start
~~~
